# Voxel Sandbox

Singleplayer/offline Voxel-Spiel im Browser.

## Spielen ohne Server

1. Einmal bauen: `npm run build`
2. Danach `index.html` direkt im Browser öffnen.

`app.html` ist nur die Build-Vorlage für Vite. Zum Spielen brauchst du keinen lokalen Server.

Die Welt wird nicht gespeichert. Reload oder Reset erzeugt wieder dieselbe frische Landschaft.
